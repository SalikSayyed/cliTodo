# Todo Command Line Application

## Very Simple and Easy to use Todo Command Line application.

### How to use ?

- #### npm install -g cliTodo
- #### todo --help
- #### t -h

### Both t/todo can be used

- ##### t l => listing todos
- ##### t a => add todo
- ##### t -h =>todo help
- ##### t md =>make todo done
- ##### t mu =>undone a todo
- ##### t rm =>remove todo
- ##### t ra =>remove all todos
